系统分析师
工程研发中心
5
深圳
2016-03-26

岗位职责：

1、研究所负责产品的业务规则，掌握该类业务的最新发展动态，完善产品功能；

2、参与各产品/项目的业务需求分析工作，输出相关说明文档；

3、进行需求讲解，与开发团队共同完成产品功能，实现方案设计、架构设计；

4、解决开发人员和测试人员遇到的需求问题，参与需求评审、代码评审、测试用例评审；

5、跟进产品需求开发进展，管理产品/项目需求列表，排定优先级；

6、参与产品规划，制定产品发展战略

任职要求：

1、211、985院校全日制本科及以上学历，计算机、金融相关专业，特别优秀者可以放宽；

2、2年以上系统开发/系统分析/需求分析相关经验，熟悉金融行业相关业务知识和系统开发过程；

3、具有良好的沟通、理解、分析能力；

4、有数据库的基础知识并熟悉主流数据库oracle，ms server sql使用者优先；

5、有销售管理、份额登记、网上交易、互联网金融、客户服务、资金管理、支付平台等系统相关经验者优先。

工程研发中心
5
深圳
2016-03-24

1、负责公司品牌塑造、产品品牌塑造等相关的视觉设计（包括：VI、广告、网页、动画以及其他推广媒介的设计）；

2、参与经营和管理品牌，制定品牌视觉规范；

3、参与工作流程和方法梳理；

任职要求：

1、痴迷于创意，热心于品牌视觉设计,善于把握潮流动向；

2、在包括平面设计（广告、VI、宣传品等）和包装设计等领域有较强的专业能力，具备从创意构思到画面执行的能力；

3、有一定的策划和文案撰写能力；

4、具有广告公司或设计公司经验者优先考虑；

5、对IT业界有一定的理解和认识，能够很好的理解客户需求；

6、具备强的执行力，良好的合作态度、沟通能力及团队精神，并富有工作激情和责任感；

7、精通Photoshop、Illustartor等设计工具，懂Flash更佳；

8、美术、设计或相关专业本科以上学历（优秀人才，专业和学历可适当放宽）；

9、请附上个人近期作品若干（如果网页上不够空间上传，请将作品发至邮箱：hr@iuni.com）。
